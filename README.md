# ShaolinVsWutangVR
Shaolin Vs. Wutang 2 VR mod

Steam Page: https://store.steampowered.com/app/1217050/Shaolin_vs_Wutang_2/

# Instructions:
- Step 1: Extract Bepinx version 5.4.21 into the game directory (BepInEx_x64_5.4.21.0.zip) https://github.com/BepInEx/BepInEx/releases/tag/v5.4.21
- Step 2: Run the game one time to set up BepinEx.
- Step 3: Extract Mod into game directory

# Current Status:
- Mostly tested Arcade mode and training mode.
- Pause menu might have issues.
- Select button on gamepad for alternate camera view on some levels
